## ğŸš€ Getting Started

### Installation

#### Option A: Automated Setup (Recommended)

1. **Install Minecraft Forge 14.23.5.2855** for Minecraft 1.12.2
2. **Clone or download this repository:**
   ```powershell
   git clone https://github.com/huskywingspan/ENERGYCRAFT.git
   cd ENERGYCRAFT
   ```
3. **Run the setup script** to download mods:
   ```powershell
   # Windows PowerShell
   .\setup.ps1
   ```
4. **Point your launcher** to this folder as the instance directory
5. **Launch and play!**

#### Option B: Manual Installation

1. **Download the latest release** from [GitHub Releases](https://github.com/huskywingspan/ENERGYCRAFT/releases)
2. **Download `energycraft-mods.zip`** from the release assets
3. **Extract** the repo contents to your Minecraft instance folder
4. **Extract `energycraft-mods.zip`** contents into the `mods/` folder
5. **Launch with Forge 14.23.5.2855**
